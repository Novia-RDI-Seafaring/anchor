Anchor UI

RAG-Powered Knowledge Base Assistant

Welcome to Anchor UI

Anchor UI is an open-source, RAG (Retrieval-Augmented Generation) powered knowledge base assistant that helps you interact with your documents through an intelligent chat interface. Built with Next.js, FastAPI, and PostgreSQL with pgvector, Anchor UI provides a modern, efficient way to manage and query your knowledge base.

Quick Start

Get up and running in minutes:

  1. Install dependencies: npm install
  2. Set up your environment variables (see Getting Started)
  3. Start the application: npm run dev
  4. Open your browser to http://localhost:3000

Key Features

📚 Knowledge Base

Upload documents (PDF, DOCX, TXT, MD, HTML) or add URLs to build your knowledge base

💬 Intelligent Chat

Ask questions and get answers based on your documents using RAG technology

🔍 Semantic Search

Find relevant information using vector similarity search powered by pgvector

⚙️ Flexible Configuration

Customize AI models, temperature, tokens, and enable MCP tools

🌙 Dark Mode

Comfortable viewing in both light and dark themes

📊 Document Management

View, filter, delete, and re-process documents with ease

Technology Stack

  • Frontend: Next.js 15, React 19, TypeScript, Tailwind CSS, CopilotKit
  • Backend: FastAPI, PydanticAI, AG-UI Protocol
  • Database: PostgreSQL with pgvector extension
  • AI/ML: OpenAI/Azure OpenAI (embeddings + LLM), Docling (document processing)
  • Deployment: Uvicorn (ASGI server)
