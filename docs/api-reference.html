API Reference

Base URL

All API endpoints are served from the FastAPI backend:

http://localhost:8001

Agent Endpoint

POST /agent

Purpose: AG-UI Protocol endpoint for agent communication

Protocol: AG-UI Protocol (HTTP-based)

Usage: Used by CopilotKit via HttpAgent

Response: Streaming agent responses

Document Management APIs

POST /api/documents/upload

Purpose: Upload and process a document

Request:

Content-Type: multipart/form-data
Body: file (binary)

Response:

{
  "success": true,
  "document": {
    "document_id": "abc123",
    "filename": "example.pdf",
    "status": "processed",
    "chunks": 42
  }
}

Supported Formats: PDF, DOCX, TXT, MD, HTML

POST /api/documents/url

Purpose: Add a URL to the knowledge base

Request:

Content-Type: application/json
Body: {
  "url": "https://example.com/article"
}

Response:

{
  "success": true,
  "document": {
    "document_id": "def456",
    "filename": "article.html",
    "status": "processed"
  }
}

GET /api/documents

Purpose: List all documents

Response:

{
  "success": true,
  "documents": [
    {
      "id": 1,
      "document_id": "abc123",
      "filename": "example.pdf",
      "source_type": "file",
      "chunk_count": 42,
      "status": "processed",
      "created_at": "2024-01-01T00:00:00"
    }
  ]
}

DELETE /api/documents/{document_id}

Purpose: Delete a specific document

Parameters:

  • document_id (path): Document ID to delete

Response:

{
  "success": true
}

POST /api/documents/reingest

Purpose: Re-process all documents

Response:

{
  "success": true,
  "processed": 10,
  "failed": 0,
  "errors": []
}

DELETE /api/documents/reset

Purpose: Reset (clear) the entire knowledge base

⚠️ Warning: This permanently deletes all documents and chunks

Response:

{
  "success": true,
  "documents_deleted": 10,
  "chunks_deleted": 420
}

Search APIs

POST /api/search

Purpose: Search the knowledge base

Request:

Content-Type: application/json
Body: {
  "query": "search terms",
  "top_k": 5,
  "document_id": "abc123"  // optional
}

Response:

{
  "success": true,
  "results": [
    {
      "content": "chunk text...",
      "filename": "example.pdf",
      "similarity": 0.85,
      "metadata": {}
    }
  ]
}

Statistics APIs

GET /api/stats

Purpose: Get knowledge base statistics

Response:

{
  "success": true,
  "total_documents": 10,
  "processed_documents": 10,
  "total_chunks": 420,
  "status": "connected"
}

Active Document APIs

GET /api/active-document

Purpose: Get the currently active document filter

Response:

{
  "document_id": "abc123"  // or null for "all"
}

POST /api/active-document

Purpose: Set the active document filter

Request:

Query Parameters:
  document_id: "abc123"  // or omit for "all"

Response:

{
  "success": true,
  "document_id": "abc123"
}

Health Check

GET /health

Purpose: Health check endpoint

Response:

{
  "status": "healthy"
}

Error Responses

All endpoints may return error responses:

{
  "detail": "Error message"
}

Status Codes:

  • 400 - Bad Request
  • 404 - Not Found
  • 500 - Internal Server Error

Authentication

Currently, Anchor UI does not implement authentication. All endpoints are publicly accessible. For production use, consider implementing:

  • API key authentication
  • JWT tokens
  • OAuth2

CORS

The backend is configured with CORS middleware allowing all origins. For production, restrict to specific domains:

allow_origins=["https://yourdomain.com"]

Rate Limiting

Currently not implemented. Consider adding rate limiting for production use.

API Documentation

FastAPI automatically generates interactive API documentation:

  • Swagger UI: http://localhost:8001/docs
  • ReDoc: http://localhost:8001/redoc

Example Usage

Upload a Document

const formData = new FormData();
formData.append('file', fileInput.files[0]);

const response = await fetch('http://localhost:8001/api/documents/upload', {
  method: 'POST',
  body: formData
});

const data = await response.json();
console.log(data);

Search Knowledge Base

const response = await fetch('http://localhost:8001/api/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    query: 'maximum operating temperature',
    top_k: 5
  })
});

const data = await response.json();
console.log(data.results);

Get Statistics

const response = await fetch('http://localhost:8001/api/stats');
const data = await response.json();
console.log(`Total documents: ${data.total_documents}`);
console.log(`Total chunks: ${data.total_chunks}`);
