Command Line Interface

Overview

Anchor UI provides several command-line options for development, building, and running the application. This guide covers all available commands and their options.

Frontend Commands

Development

npm run dev

Starts both frontend and backend in development mode:

  • Frontend: Next.js dev server on port 3000
  • Backend: FastAPI with uvicorn on port 8001
  • Hot reload enabled for both
  • Uses concurrently to run both services

Individual Services

npm run dev:ui

Start only the frontend (Next.js):

  • Runs on http://localhost:3000
  • Hot module replacement enabled
  • Requires backend to be running separately
npm run dev:agent

Start only the backend (FastAPI):

  • Runs on http://localhost:8001
  • Auto-reload on code changes
  • Log level: warning (reduced verbosity)

Production Build

npm run build

Build the Next.js application for production:

  • Optimizes React components
  • Minifies JavaScript and CSS
  • Generates static assets
  • Output in .next directory

Production Server

npm start

Start the production server:

  • Runs built Next.js application
  • Requires npm run build first
  • Optimized for performance
  • No hot reload

Linting

npm run lint

Run ESLint to check code quality:

  • Checks TypeScript/JavaScript files
  • Reports syntax errors and warnings
  • Follows Next.js recommended rules

Backend Commands

Running the Backend

The backend can be run directly with uvicorn:

cd backend
uv run uvicorn main:app --reload --port 8001 --log-level warning

Uvicorn Options

Option Description Default
--reload Enable auto-reload on code changes Enabled in dev
--port Port number to bind 8001
--host Host to bind to 0.0.0.0
--log-level Logging level (debug, info, warning, error) warning

Python Package Management

Anchor UI uses uv for Python dependency management:

cd backend
uv sync

Install all dependencies from pyproject.toml

uv add package-name

Add a new dependency

uv remove package-name

Remove a dependency

Database Commands

PostgreSQL Setup

# Connect to PostgreSQL
psql -U postgres

# Create database
CREATE DATABASE anchor_ui;

# Connect to database
\c anchor_ui

# Enable pgvector extension
CREATE EXTENSION vector;

Database Migrations

Anchor UI automatically creates tables on first connection. No manual migrations needed.

Development Workflow

Initial Setup

# Install frontend dependencies
npm install

# Install backend dependencies
cd backend
uv sync

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Set up database
# (See Database Commands above)

# Start development servers
cd ..
npm run dev

Daily Development

# Start everything
npm run dev

# Or start separately
npm run dev:ui    # Terminal 1
npm run dev:agent # Terminal 2

Production Deployment

Build Process

# Build frontend
npm run build

# Backend doesn't need building (Python)
# But ensure dependencies are installed:
cd backend
uv sync --production

Running in Production

# Frontend
npm start

# Backend (use process manager like PM2 or systemd)
cd backend
uv run uvicorn main:app --host 0.0.0.0 --port 8001

Debugging Commands

Check Backend Health

curl http://localhost:8001/health

View Backend Logs

Logs appear in the terminal where the backend is running. For more verbose logging:

cd backend
uv run uvicorn main:app --reload --port 8001 --log-level debug

Database Debugging

# Connect to database
psql -U postgres -d anchor_ui

# Check documents
SELECT * FROM documents;

# Check chunks
SELECT COUNT(*) FROM chunks;

# Check vector extension
SELECT * FROM pg_extension WHERE extname = 'vector';

Common Issues

Port Already in Use

# Find process using port 3000
lsof -i :3000
# or
netstat -ano | findstr :3000  # Windows

# Kill process
kill -9 <PID>

Dependencies Out of Sync

# Frontend
rm -rf node_modules package-lock.json
npm install

# Backend
cd backend
rm -rf .venv
uv sync

Build Errors

# Clear Next.js cache
rm -rf .next
npm run build

Scripts Reference

All scripts are defined in package.json:

Script Command Description
dev concurrently "npm:dev:ui" "npm:dev:agent" Run both frontend and backend
dev:ui next dev Run Next.js dev server
dev:agent cd backend && uv run uvicorn main:app --reload --port 8001 --log-level warning Run FastAPI backend
build next build Build for production
start next start Start production server
lint next lint Run ESLint
