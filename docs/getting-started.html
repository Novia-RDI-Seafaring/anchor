Getting Started

Prerequisites

Before you begin, ensure you have the following installed:

  • Node.js (v18 or higher) - Download
  • Python (v3.12 or higher) - Download
  • PostgreSQL (v12 or higher) with pgvector extension - Download
  • uv (Python package manager) - Installation Guide

Installation

1. Clone the Repository

git clone https://github.com/your-org/anchor-ui.git
cd anchor-ui

2. Install Frontend Dependencies

npm install

3. Install Backend Dependencies

cd backend
uv sync

Configuration

Database Setup

First, set up PostgreSQL with the pgvector extension:

# Connect to PostgreSQL
psql -U postgres

# Create database
CREATE DATABASE anchor_ui;

# Connect to the database
\c anchor_ui

# Enable pgvector extension
CREATE EXTENSION vector;

Environment Variables

Create a .env file in the backend directory:

# Database Configuration
PGVECTOR_HOST=localhost
PGVECTOR_PORT=5432
PGVECTOR_DB=anchor_ui
PGVECTOR_USER=postgres
PGVECTOR_PASSWORD=your_password
PGSSLMODE=disable

# API Configuration
HOST=0.0.0.0
PORT=8001
RELOAD=true

# OpenAI Configuration (for embeddings and LLM)
OPENAI_API_KEY=your_openai_api_key
# OR for Azure OpenAI:
# AZURE_OPENAI_API_KEY=your_azure_key
# AZURE_OPENAI_ENDPOINT=https://your-resource.openai.azure.com/

# LLM Model (optional, defaults to gpt-4o-mini)
LLM_MODEL=gpt-4o-mini

# RAG Configuration
CHUNK_SIZE=512
CHUNK_OVERLAP=50
TOP_K_RESULTS=5
SIMILARITY_THRESHOLD=0.3

# Document Storage
UPLOADS_DIR=../uploads

# Logfire (Observability - optional)
LOGFIRE_TOKEN=your_logfire_token

Frontend Environment Variables

Create a .env.local file in the root directory:

# Backend URL (default: http://localhost:8001)
NEXT_PUBLIC_BACKEND_URL=http://localhost:8001

Running the Application

Development Mode

From the root directory, run:

npm run dev

This command starts both the frontend (Next.js on port 3000) and backend (FastAPI on port 8001) concurrently.

Individual Services

You can also run services separately:

  • Frontend only: npm run dev:ui
  • Backend only: npm run dev:agent

Verification

Once the application is running:

  1. Open your browser to http://localhost:3000
  2. You should see the Anchor UI interface
  3. Check the backend health: http://localhost:8001/health

First Steps

  1. Upload a Document: Click the paperclip icon in the chat input or go to Settings → Knowledge Base Management
  2. Wait for Processing: Documents are automatically processed and embedded
  3. Start Chatting: Ask questions about your uploaded documents

Troubleshooting

Database Connection Issues

  • Verify PostgreSQL is running: pg_isready
  • Check connection credentials in .env
  • Ensure pgvector extension is installed: SELECT * FROM pg_extension WHERE extname = 'vector';

Backend Not Starting

  • Check Python version: python --version (should be 3.12+)
  • Verify dependencies: cd backend && uv sync
  • Check port 8001 is not in use

Frontend Not Starting

  • Check Node.js version: node --version (should be 18+)
  • Clear cache: rm -rf node_modules .next && npm install
  • Check port 3000 is not in use

Document Upload Fails

  • Ensure uploads directory exists and is writable
  • Check file format is supported (PDF, DOCX, TXT, MD, HTML)
  • Verify OpenAI API key is set correctly
