Features

Core Features

📚 Knowledge Base Management

Anchor UI provides comprehensive document management capabilities:

  • Multi-format Support: Upload PDF, DOCX, TXT, MD, and HTML files
  • URL Import: Add web pages directly to your knowledge base
  • Automatic Processing: Documents are automatically converted, chunked, and embedded
  • Document Filtering: Focus queries on specific documents or search across all
  • Bulk Operations: Re-process all documents or reset the knowledge base
  • Statistics Dashboard: View document and chunk counts

💬 Intelligent Chat Interface

Natural language interaction with your knowledge base:

  • RAG-Powered: Responses based on your uploaded documents
  • Conversation History: Multiple conversations with persistent history
  • Context Controls: Toggle RAG context and web search
  • Real-time Streaming: See responses as they're generated
  • Source Citations: View which documents informed each response
  • Minimizable Panel: Collapse chat to focus on workspace

🔍 Semantic Search

Advanced vector-based search capabilities:

  • Vector Embeddings: Uses OpenAI text-embedding-3-large (3072 dimensions)
  • Similarity Scoring: Cosine similarity for relevance ranking
    • Configurable Top-K: Adjust number of results retrieved
  • Threshold Filtering: Filter results by similarity threshold
  • Document-Specific Search: Query within a single document
  • Fast Retrieval: HNSW index for efficient similarity search

⚙️ Flexible Configuration

Customize your AI assistant to your needs:

  • Chat Model Selection: Choose from available LLM models (OpenAI, Azure, Ollama) directly in the header
  • Embedding Model: Configure specialized embedding models via Settings
  • Temperature Control: Adjust creativity vs. consistency (0.0-1.0)
  • Token Limits: Set maximum response length
  • System Prompts: Define assistant behavior and instructions
  • MCP Integration: Enable Model Context Protocol tools
  • RAG Parameters: Configure chunk size, overlap, and top-k results

Document Processing Features

📄 Docling Integration

Advanced document conversion:

  • PDF Processing: Extract text, tables, and structure
  • Table Preservation: Maintains table formatting
  • Markdown Output: Clean, structured markdown conversion
  • Fast Mode: Optimized for speed
  • Fallback Handling: Graceful degradation for unsupported formats

✂️ Intelligent Chunking

Smart text segmentation:

  • Configurable Size: Default 512 tokens, customizable
  • Overlap Strategy: 50-token overlap for context preservation
  • Metadata Preservation: Chunk-level metadata tracking
  • Index Tracking: Maintains chunk order and relationships

👁️ Visual Page Previews

See exactly where answers come from:

  • High-Fidelity Rendering: View actual PDF pages, not just text
  • Answer Highlighting: Red bounding boxes show the exact paragraphs used
  • Contextual Verification: Instantly verify AI claims against the source document
  • Seamless Integration: Appears automatically when relevant to your query
