User Guide

Overview

Anchor UI provides an intuitive interface for interacting with your knowledge base through natural language. This guide will help you make the most of all features.

Interface Overview

Main Components

  • Sidebar: Navigation, conversations, and settings access
  • Header: Document selector and model configuration
  • Workspace: Displays retrieved documents and agent responses
  • Chat Panel: Interactive conversation interface

Working with Documents

Uploading Documents

You can upload documents in two ways:

Method 1: From Chat Input

  1. Click the paperclip icon (📎) in the chat input area
  2. Select one or more files (PDF, DOCX, TXT, MD, HTML)
  3. Wait for upload confirmation - files are automatically processed

Method 2: From Settings

  1. Click "Settings" in the sidebar
  2. Navigate to "Knowledge Base Management"
  3. Click "CHOOSE FILES" and select your documents
  4. Click "Upload" to process

Adding URLs

  1. Go to Settings → Knowledge Base Management
  2. Enter a URL in the "Add URL to Knowledge Base" field
  3. Click "Add URL to Knowledge Base"
  4. The webpage content will be fetched and processed

Selecting Active Document

To focus queries on a specific document:

  1. Use the document dropdown in the header
  2. Select "All Documents" or a specific document
  3. All subsequent queries will be filtered to the selected document

Using the Chat Interface

Starting a Conversation

  1. Click "New Chat" in the sidebar or use the floating chat button
  2. Type your question in the input area
  3. Press Enter or click the send button (↑)
  4. The assistant will query your knowledge base and respond

Context Controls

The chat input includes context control buttons:

  • Context On/Off: Toggle RAG (knowledge base) context
  • Web: Enable web search capabilities (coming soon)

Best Practices for Queries

  • Be specific: "What is the maximum operating temperature?" vs "Tell me about temperature"
  • Ask follow-ups: Build on previous questions in the conversation
  • Reference documents: "According to the manual, how do I..."
  • Use natural language: No need for special syntax or commands

Managing Conversations

Creating New Conversations

  • Click "New Chat" button in the sidebar
  • Or use the keyboard shortcut (if configured)
  • Each conversation maintains its own history

Renaming Conversations

  1. Hover over a conversation in the sidebar
  2. Click the edit icon (✏️)
  3. Type the new name
  4. Press Enter or click the checkmark (✓)

Deleting Conversations

  1. Hover over a conversation in the sidebar
  2. Click the delete icon (🗑️)
  3. Confirm deletion in the dialog

Viewing Results

Workspace View

The main workspace displays:

  • Source Documents: Cards showing which documents were retrieved
  • Active Document: The main response from the assistant
  • Retrieved Context: Chunks from the knowledge base with similarity scores

Understanding Similarity Scores

Each retrieved chunk shows a similarity score (0.0 to 1.0):

  • Higher scores (0.7+): Very relevant to your query
  • Medium scores (0.4-0.7): Moderately relevant
  • Lower scores (<0.4): Less relevant (may be filtered out)

Visual Verification

For questions answered from PDF documents, Anchor UI displays a visual confirmation:

  • Page Preview: A high-quality image of the actual page from the document.
  • Red Bounding Boxes: Highlight the specific text or section used to generate the answer.
  • Usage: Use this to instantly verify that the AI is interpreting the table, chart, or text correctly.

Settings Configuration

AI Assistant Configuration

In Settings, you can configure:

  • System Prompt: Define how the assistant behaves
  • Model Selection: Choose the LLM model
  • Temperature: Control response creativity (0.0-1.0)
  • Max Tokens: Limit response length
  • MCP Tools: Enable/disable Model Context Protocol tools

Knowledge Base Management

Manage your documents:

  • View Statistics: See total documents and chunks
  • Upload Files: Add new documents
  • Add URLs: Import web content
  • Delete Documents: Remove unwanted documents
  • Reingest: Re-process all documents
  • Reset: Clear entire knowledge base (dangerous!)

Tips and Tricks

Maximizing Search Quality

  • Upload well-structured documents for better chunking
  • Use descriptive filenames - they appear in search results
  • Break large documents into logical sections

Improving Responses

  • Enable "Context On" for document-based queries
  • Select a specific document when asking about it
  • Ask follow-up questions to refine results

Managing Storage

  • Regularly review and delete unused documents
  • Monitor chunk count in document list
  • Use "Reingest" if document processing seems incorrect

Common Workflows

Research Workflow

  1. Upload research papers or documents
  2. Start a new conversation
  3. Ask specific questions about the content
  4. Review retrieved chunks for sources
  5. Save important conversations

Documentation Q&A

  1. Upload technical documentation
  2. Select the specific document in header
  3. Ask "How do I..." or "What is..." questions
  4. Get answers with source citations
